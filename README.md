# 📝 Text Analysis using NLP

A collection of Natural Language Processing (NLP) projects covering **Sentiment Analysis**, **Text Summarization**, and **Language Translation** — built with Python and modern NLP libraries.

---

## 📁 Repository Structure

| File | Description |
|------|-------------|
| `Sentiment Analyser.ipynb` | Jupyter notebook for sentiment classification of text input |
| `sentiment_analyzer.html` | Interactive web-based sentiment analyzer |
| `Text Summarization - News Article Summarizer.ipynb` | Automatic summarization of news articles using NLP |
| `Language Translation - Multi-language Support.ipynb` | Multi-language text translation using NLP models |

---

## 🚀 Projects Overview

### 1. 🎭 Sentiment Analyser
Analyzes the sentiment of a given text and classifies it as **Positive**, **Negative**, or **Neutral**.

- Text preprocessing (tokenization, stopword removal)
- Sentiment scoring and classification
- Visual output via an interactive HTML interface (`sentiment_analyzer.html`)

### 2. 📰 Text Summarization — News Article Summarizer
Automatically generates concise summaries from long-form news articles.

- Extractive and/or abstractive summarization
- Input: raw news article text or URL
- Output: shortened summary preserving key information

### 3. 🌐 Language Translation — Multi-language Support
Translates text across multiple languages using pre-trained NLP models.

- Supports multiple source and target languages
- Leverages transformer-based translation models
- Clean input/output interface within the notebook

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Libraries:** `transformers`, `NLTK`, `TextBlob`, `scikit-learn`, `pandas`, `matplotlib`
- **Environment:** Jupyter Notebook
- **Frontend:** HTML/CSS/JavaScript (for sentiment analyzer UI)

## ⚙️ Getting Started

### Prerequisites

```bash
pip install transformers nltk textblob scikit-learn pandas matplotlib
```

### Run the Notebooks

1. Clone the repository:
   ```bash
   git clone https://github.com/Muthukumaran2098/Text-Analysis-using-NLP.git
   cd Text-Analysis-using-NLP
   ```

2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

3. Open any `.ipynb` file and run all cells.

### Run the Sentiment Analyzer (Web)

Open `sentiment_analyzer.html` directly in your browser — no server required.
